Alterações do Código

1. O primeiro erro é a div do cabeçalho, ela deve ser chamar "Header", segundo que depois dele fornecer o id cabeçalho, é melhor utilizar um arquivo CSS e manipular o estilo do cabeçalho por lá usando a id registrada.

2. Dentro da TAG h1, deve só haver o texto, não se coloca IMG dentro, e por mais que a tentativa fosse de escrever perto da imagem, existem outras maneiras. a IMG e o h1 também devem ser separados por div.

3.A segunda div deveria se chamar MAIN, devido a ser o conteúdo principal como foi apelidado no ID. Após declarar a div tem um texto, que não está dentro de nenhuma TAG, por mais que o navegador vá exibir está totalmente incorreto.

4. A tag br é útil para espaçar, porém é melhor abrir duas tags diferentes espaçar elas dentro do CSS.

5. A tag b está sendo fechada antes da tag i que foi aberta depois, está incorreto. O mesmo erro de falta de tag é repetido com o texto fora do b e i e depois a div é fechada.
